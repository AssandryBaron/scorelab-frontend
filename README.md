
  # Sistema de tareas domésticas

  This is a code bundle for Sistema de tareas domésticas. The original project is available at https://www.figma.com/design/g1fwVFlyInhozoii6Thw1q/Sistema-de-tareas-dom%C3%A9sticas.
  

  ## Running the code
Corran 'npm install' en la terminal del IDE, para que puedan tener todas las librerias de react.

Ya esta conectado el registro y el login con la api del back que les mande. Testeen y me cuentan como les va. No sobra decir que primero conecten con la base de datos que ustedes crean, metan sus credenciales en la carpeta del back como les dije en el readme, y despues de tener eso seteado, ahi si testear back y front juntos. Cualquier cosa me avisan.
  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
